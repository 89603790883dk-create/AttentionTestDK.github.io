# Go/NoGo Attention Test

Простой веб-тест на внимание (вариант Go/NoGo) для демонстрационных целей.

## Как запустить тест

1. Открой страницу теста по ссылке  
   `https://89603790883dk-create.github.io/DKtest.github.io-test/test.html`
2. Нажми кнопку **«Начать тест»**.
3. Смотри на буквы в центре экрана.

## Правила

- Нажимай кнопку **«ОТВЕТИТЬ»** или клавишу **пробел**, когда видишь букву **X**.
- Не нажимай на другие буквы (**Y, V, K, W**).
- На реакцию даётся **1.5 секунды**.
- Старайся реагировать как можно **быстрее и точнее**.

## Результаты

После завершения теста отображаются:

- Общая точность в процентах.
- Среднее время реакции (мс) по правильным ответам.
- Количество правильных ответов из 18 целевых стимулов.
- Количество ошибок (нажатий на нецелевые буквы).
